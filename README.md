# FusionNet

[FusionNet: A deep fully residual convolutional neural network for image segmentation in connectomics](https://arxiv.org/abs/1612.05360)

## Preprocess data
~~~
python3 -m preprocess
~~~
## Train Model
~~~
python3 -m train
~~~
## Test Model
~~~
python3 -m test
~~~
## Deploy Model
~~~
python3 -m deploy
~~~
## Evaluate Model
~~~
python3 -m eval
~~~
